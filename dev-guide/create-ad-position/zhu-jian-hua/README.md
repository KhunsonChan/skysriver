# 组件化方式接入广告位

{% hint style="warning" %}
目前，天幕广告位接入的方式有两种：  
1、组件化的方式接入（仅支持cocos、laya引擎）  
2、api方式接入。
{% endhint %}

若游戏引擎符合要求，推荐通过使用组件化进行接入，这将会大大提升接入的效率。
